# 多项选择

- 支持隐藏选项以避免潜在的提示
- 支持单选与多选
- 选项在展示答案后会恢复到展示前的顺序
- 明确的答案标记
- 支持键盘快捷键：使用 Ctrl+1/2/3... 快速选择对应的选项 A/B/C...

## 字段

注意：当所有选项均为空时，模板会作为基础问答题使用

| 字段名        | 说明                                                                                                                                                                                                                           |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| question      | 题干，支持加粗、公式等富文本格式                                                                                                                                                                                               |
| optionA...F   | 选项内容；未填写的选项不会展示，同样支持多种格式                                                                                                                                                                               |
| answer        | 答案；单选题填写正确答案的大写字母，如 A；多选题填写所有正确答案字母，如 ABC                                                                                                                                                   |
| note          | 可填写详细解析、备注等                                                                                                                                                                                                         |
| noteA...F     | 每个选项的备注/解析                                                                                                                                                                                                            |
