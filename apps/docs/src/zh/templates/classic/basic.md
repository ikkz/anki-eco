# 基础问答

一个简单的问答模板

---

[[toc]]

## 字段

| 字段名   | 说明                                            |
| -------- | ----------------------------------------------- |
| question | 题干，支持加粗、公式等富文本格式                 |
| answer   | 答案，同样支持多种格式                          |
| note     | 解析/备注等                                      |

## 预览与下载
