# 输入题

输入题模板。类似 Anki 的 type 字段，但支持任意数量的输入框。

使用时，将需要填空的文本用双大括号包裹，例如 <span v-pre>`{{text}}`</span>。支持多个填空，也支持图片/公式填空。

AnkiDroid 用户需在高级设置中开启 “Type answer into the card”。
