# Ordering

Drag and drop fragments to arrange them into the correct order.

> [!TIP]
> It is best to disable all swipe gesture controls in Anki's review settings.

---

[[toc]]

## Fields

Notes for `items`

- Fragments can be separated by `,,` or by new lines
- Fragments are shuffled to enhance active recall

An example:

```
The quick,,brown,,fox
jumps over,,the lazy dog
```

| Field name | Description                                                                                                   |
| ---------- | ------------------------------------------------------------------------------------------------------------- |
| question   | This is the stem of the question. It supports various content formats in Anki, including bold, formulas, etc. |
| items      | The fragments to arrange                                                                                      |
| note       | You can fill in detailed explanations, notes, etc., here.                                                     |
